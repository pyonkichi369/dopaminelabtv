## 何をしているか

30〜50代の疲れた社会人に向けて、脳科学の研究を日本哲学と接続しながら届けるメディアです。

難しい論文を読みやすく翻訳するのではなく、**脳で起きていることと、日本人が何千年もかけて経験的に発見したことが、同じことを指している**——その構造を示すことを目的としています。

---

## コンテンツ公式

すべての記事・コンテンツは **「ドーパミン × X」** の掛け合わせで構成されます。

**ドーパミンは常に主語。** 他のホルモンや哲学概念・現代問題は、「ドーパミンとの関係」として登場します。

### 掛け合わせの例

| ドーパミン × 神経化学 | ドーパミン × 日本哲学 | ドーパミン × 現代問題 |
|---------------------|---------------------|---------------------|
| コルチゾール（疲労）  | 無為（むい）         | スマホ依存           |
| オキシトシン（孤独）  | 間（ま）             | 燃え尽き症候群        |
| セロトニン（運動）    | 整う（ととのう）      | ADHD・集中力         |
| ノルエピネフリン      | 渇愛（かつあい）      | 先延ばし             |

### コンテンツ設計の原則

1. **神経科学が先、日本哲学が補完** — 研究から始め、哲学で深める
2. **宗教ではなく経験則** — 禅・仏教の概念は「2,500年の実践データ」として扱う
3. **処方ではなく構造の理解** — 「こうしなさい」ではなく「なぜこうなるか」を示す
4. **PubMed引用のみ** — すべての科学的主張は検証可能なソースに基づく

## 記事フォーマット

```yaml
---
title: "記事タイトル"
date: "YYYY-MM-DD"
excerpt: "1〜2文の要約"
tags: ["神経科学", "ドーパミン", ...]
readTime: 7
tldr:
  - "要点1"
  - "要点2"
  - "要点3"
sources:
  - name: "著者 et al., Journal, Year"
    url: "https://pubmed.ncbi.nlm.nih.gov/PMID/"
---
```

すべての記事に TL;DR（3点）と PubMed ソースを必須とします。
